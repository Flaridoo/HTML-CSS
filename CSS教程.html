一、CSS基础

（一）CSS基础语法

select {
  property: value
}
例如：
h1 {
  color: red;
  font-size: 14px;
}

①属性大于一个，各个属性要用分号隔开。

例如：
p {
  font-family: "sans serif";

普通布局 根据先后位置确定排布
浮动布局 像浮在水上一样，具有灵活性
绝对布局 根据坐标数值精确定位布局

clip 设置元素显示的形状
vertical-align 设置元素垂直对其方式
z-index 设置元素的堆叠上下顺序
